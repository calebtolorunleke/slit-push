Frontend Development

JavaScript CSS

Frontend development is the development of graphcical user interface of a website, through HTML, CSS and JavaScript, so that users can view and interact with the website.

  • HTML: The HyperText Markup Language or HTML is the standard markup language for documents designed to be displayed in a web browser.

  • CSS: Cascading Style Sheet (CSS) is a style sheet language used for decorating the presentation of a document or written in a markup language such as HTML or XML.

    1. Pure CSS

    2. Bootstrap (Framework)

    3. Tailwind (Framework)

    4. Sass (Preprocessor)


  • JavaScript: JavaScript often abbreviated JS, is a programming language that is ome of the core technologies of the World Wide Web, alngside HTML and CSS. It is use to add functionality in the website.

    1. Vanila JS

    2. jQuery (library)

    3. React (Framework)

    4. Angular (Framework)

    5. Vue (Framework)

    6. Typescript (Preprocessor)
